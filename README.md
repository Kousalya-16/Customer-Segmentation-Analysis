# Customer Segmentation Analysis

## Project Overview
This project focuses on analyzing customer shopping behavior to identify high-value customer segments and revenue-driving patterns using interactive Power BI dashboards.

## Tools Used
- Power BI  
- Excel  

## Analysis Performed
- Customer segmentation based on age, gender, purchase category, and payment method  
- Revenue and category-level performance analysis  
- Identification of high-value customer groups  

## Key Insights
- Specific age and gender segments contribute higher revenue  
- Certain product categories consistently outperform others  
- Payment methods influence purchase frequency and spending behavior  

## Business Impact
The insights from this analysis help businesses design targeted marketing strategies, improve customer engagement, and optimize revenue growth.
